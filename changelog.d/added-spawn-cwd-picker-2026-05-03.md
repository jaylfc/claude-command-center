- "+ New session" now exposes a folder dropdown above the input box so you pick where the new session will land before submitting. Default = the active folder filter (when narrowed to one repo) or the server's REPO_ROOT (when filter is "All"); selection persists in localStorage. Previously `spawnFromInlineInput()` silently used the server's REPO_ROOT regardless of which folder you were viewing — invisible from the All-repos UI and a frequent source of "wrong-folder spawn" surprises.
